## API

### Icon props

| 属性         | 说明                                   | 类型                          | 默认值  |
| ------------ | -------------------------------------- | ----------------------------- | ------- |
| type         | icon类型                               | string                        |         |
| className    | class                                  | string                        |         |
| rotate       | 图标旋转角度（IE9 无效）               | number                        |         |
| spin         | 是否有旋转动画                         | string/ horizontal、 vertical | boolean |
| style        | 设置图标的样式，例如 fontSize 和 color | CSSProperties                 |         |
| twoToneColor | 仅适用双色图标。设置双色图标的主要颜色 | string (十六进制颜色)         |         |
